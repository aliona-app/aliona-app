# Офлайн-читалка Алёны

Полностью локальная PWA-читалка на React + Vite.

## Что изменено в этой версии

- приложение переведено на **HashRouter**, чтобы GitHub Pages и установленная PWA не ломались на внутренних экранах
- главный экран обновлён под персональный текст:
  - **Офлайн-читалка Алёны**
  - **Premium Reader**
  - **С любовью для тебя ❤️**
- убрана зависимость от внешних Google Fonts, чтобы интерфейс был максимально локальным
- service worker настроен агрессивнее для локального app shell
- исправлена логика ридера:
  - убрана переинициализация FlowReader на каждый прогресс
  - убрана лишняя переинициализация EPUB по текущей позиции
  - добавлен более надёжный back flow из книги и ридера
- улучшен UX импорта и пустой библиотеки

## Запуск

```bash
npm install
npm run dev
```

## Деплой на GitHub Pages

У проекта уже выставлен `base: /aliona-app/` под репозиторий:

`https://github.com/aliona-app/aliona-app`

Если имя репозитория изменится, обнови `base`, `start_url` и `scope` в `vite.config.js`.


Актуальный GitHub Pages URL после деплоя:

`https://aliona-app.github.io/aliona-app/`
